# Overview

As a software engineer, I am building foundational programming skills by creating simple applications that demonstrate core concepts. This project represents the starting point of my journey to develop more complex software solutions throughout this course.

This is a basic "Hello World" application written in JavaScript. The program outputs a greeting message to the console, serving as a simple introduction to setting up a development environment, working with version control, and establishing good documentation practices.

My purpose for creating this software is to establish a working development workflow that I will use throughout the semester. By completing this project, I am ensuring that my tools are properly configured and that I understand the process of creating, documenting, and sharing code through GitHub. This foundational exercise prepares me for more challenging programming projects ahead.

# Development Environment

For this project, I used Visual Studio Code as my integrated development environment (IDE). Visual Studio Code provides excellent JavaScript support with features like syntax highlighting, code completion, and integrated terminal access.

I used Git for version control and GitHub for hosting my repository. The code is written in JavaScript and runs using Node.js, which allows JavaScript to execute outside of a web browser environment.

The programming language used is JavaScript. I chose JavaScript because it is widely used, versatile, and I already have Node.js installed on my computer from previous learning experiences.
